# Matrix to Telegram Themes

Настройки работают на Matrix Element 1.12.13 для Windows 10

1. Сначала нужно создать в папке %APPDATA%\Element файл config.json со следующим содержимым:

{
    "show_labs_settings": true,
    "setting_defaults": {
        "custom_themes": []
    }
}

Данный код включает сторонние темы в Element.

2. Добавить в настройки Element во Внешний вид
